// HOW IT WORKS

3 步开始学习

01
获取邀请码

联系你的老师，获取专属学生邀请码。邀请码由教师统一发放，确保班级安全。

02
注册账号，进入 Dashboard

输入姓名、邮箱、密码与邀请码，选择班级完成注册。登录后即可看到所有工具入口和你的积分排名。

03
选择工具，开始练习！

点击任意工具卡片进入练习。老师已分配任务的工具会优先显示，完成任务积累 XP，冲上总排行榜。
